CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

1 Repository Overview

This is a personal technical knowledge repository containing documentation and notes in Chinese. The repository is organized as a static collection of markdown files covering various technology topics including algorithms, Go programming, DevOps, system design, and computer science fundamentals.

2 Directory Structure

3 Content Format Standards

All markdown files follow a consistent YAML front matter format:

3.1 Key Formatting Rules

  1. Title Format: Use brackets for category - [Algorithm] Array, [Go] Memory Management
  2. Language: Primary language is Chinese with English technical terms preserved
  3. Content Separator: Include <!--more--> for article summaries
  4. Code Blocks: Use fenced code blocks with language specified
  5. Categories: Main categories are Algorithm, Go, LLM, 摄影 (Photography)

4 Common Operations

Since this is a static documentation repository, operations typically involve:

  1. Creating new articles:

    • Follow the front matter format in 发布文章.md
    • Use appropriate category tags
    • Place in the corresponding directory

  2. Updating content:

    • Maintain consistent formatting
    • Update published date when modifying
    • Keep English technical terms in English

  3. File organization:

    • Algorithm files cover data structures and algorithmic problems
    • Golang directory has subdirectories for specific topics (packages, memory management, etc.)
    • Keep one main topic per file

5 Working with This Repository

  • No build process or package managers are used
  • All content is in markdown format with YAML front matter
  • The repository serves as a knowledge base for technical blog content
  • Files are written in Chinese but maintain English technical terminology
  • Each directory represents a major technical domain

6 Important Notes

  • This is not a code repository with executables - it’s documentation
  • Respect the Chinese/English mixing convention (Chinese explanations with English technical terms)
  • Maintain the established metadata format for consistency
  • Check 发布文章.md for the latest formatting guidelines and tag taxonomy
